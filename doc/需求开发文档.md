# AI智能面试宝典 - 需求文档

## 1. 项目概述
AI智能面试宝典是一个基于AI技术的面试辅助系统，旨在帮助用户准备面试，提高面试成功率。系统提供简历优化、自我介绍生成、智能题库、模拟面试和面试策略等功能，通过AI分析用户简历，生成个性化的面试准备内容。

### 1.1 项目目标
- 帮助用户优化简历，提高简历通过率
- 生成个性化的自我介绍，提升面试第一印象
- 提供针对性的面试题库，帮助用户准备面试问题
- 模拟真实面试场景，提升用户面试经验
- 提供面试策略分析，帮助用户制定面试策略

### 1.2 技术栈
- 前端：Vue 3 + Vite
- 后端：Flask + Python
- 数据库：MySQL
- AI服务：DeepSeek API
- 文件处理：PyMuPDF (PDF)、python-docx (DOCX)
- API客户端：OpenAI Python SDK (用于调用DeepSeek API)

## 2. 功能需求

### 2.1 首页
- 系统介绍
- 功能模块导航
- 用户引导

### 2.2 简历优化
- 简历上传（支持PDF、DOCX、TXT格式）
- 简历内容解析（使用PyMuPDF优化PDF文本提取，python-docx处理DOCX文件）
- AI智能分析简历
  - 综合评分（0-100分）
  - 诊断建议（警告/错误/建议）
  - 关键词提取（至少10个技术岗位相关关键词）
  - STAR法则重写
- 生成优化后的简历（Markdown格式）
- 简历保存和管理（支持原始简历和优化后简历的存储）

### 2.3 自我介绍
- 基于简历生成自我介绍
- 支持不同版本长度（30秒电梯演讲版、3分钟标准版、5分钟深度版）
- 支持不同语言风格（正式、活泼、专业、亲切）
- 自我介绍语音朗读
- 提词器功能
- 自我介绍导出PDF

### 2.4 智能题库
- 基于简历生成个性化题库
- 支持不同题目数量（10题、30题、50题、100题）
- 题型分布（高频必问题30%、简历深挖题25%、专业技能题25%、行为/情景题20%）
- 支持自定义话题
- 题目包含参考答案和面试官意图分析
- 题库导出功能

### 2.5 模拟面试
- 基于简历生成面试问题
- 支持不同面试风格（技术面、HR面、综合面）
- 支持不同面试模式（自由模式、限时模式）
- 支持不同面试时长
- 面试过程录制和回放
- 面试报告生成

### 2.6 面试策略
- 基于简历生成画像分析
- 提供优化方向建议
- 生成高质量的反问问题
- 支持按问题类型生成（公司发展类、团队文化类、岗位发展类、工作内容类）
- 面试策略历史记录管理

### 3.4 可扩展性需求

## 4. 用户角色

### 4.1 普通用户

## 5. 使用场景

### 5.1 场景一：用户准备面试
1. 用户上传简历，系统进行优化
2. 基于优化后的简历生成自我介绍
3. 生成个性化题库，用户进行练习
4. 进行模拟面试，获取面试反馈
5. 生成面试策略，准备最终面试

### 5.2 场景二：用户需要优化简历
1. 用户上传简历
2. 系统分析简历并提供优化建议
3. 用户查看优化后的简历
4. 用户下载优化后的简历

### 5.3 场景三：用户需要准备自我介绍
1. 用户选择自我介绍版本和风格
2. 系统生成个性化自我介绍
3. 用户可以语音朗读、使用提词器或导出PDF

# AI智能面试宝典 - 软件设计开发文档

## 1. 系统架构

### 1.1 整体架构
```
┌─────────────────────────────────────────────────────────────────┐
│                      前端应用 (Vue 3 + Vite)                     │
├─────────────────────────────────────────────────────────────────┤
│  首页  │ 简历优化 │ 自我介绍 │ 智能题库 │ 模拟面试 │ 面试策略  │
└─────────────────────────────────────────────────────────────────┘
                              │ HTTP/HTTPS
┌─────────────────────────────────────────────────────────────────┐
│                      后端服务 (Flask)                           │
├─────────────────────────────────────────────────────────────────┤
│  API路由  │ 业务逻辑 │ AI服务集成 │ 数据库操作 │ 文件服务     │
└─────────────────────────────────────────────────────────────────┘
                              │
┌─────────────────────────────────────────────────────────────────┐
│                      数据库 (MySQL)                              │
├─────────────────────────────────────────────────────────────────┤

                              │
```

- **前端**：Vue 3 + Vite + Vue Router + Axios
- **后端**：Flask + SQLAlchemy + Flask-CORS

- **AI服务**：DeepSeek API
- **文件存储**：本地文件系统

### 3.2 主要业务服务

#### 3.2.1 文件服务 (file_service.py)
- 简历文件读取（支持PDF、DOCX、TXT格式）
  - PDF：使用PyMuPDF优化文本提取
  - DOCX：使用python-docx处理
  - TXT：直接读取
- 简历内容解析和清理
- 简历文件保存（原始和优化后简历）
- 简历内容获取

#### 3.2.2 AI服务 (deepseek_service.py)
- DeepSeek API调用（使用OpenAI Python SDK）
- 简历分析（评分、诊断、关键词提取、STAR重写、优化简历生成）
- 自我介绍生成（支持不同版本和风格）
- 题库生成
- 面试问题生成
- 面试策略生成

### 3.3 API路由设计

#### 3.3.1 简历相关API
- `POST /api/resume/analyze` - 分析简历
- `POST /api/resume/get` - 获取简历数据

#### 3.3.2 自我介绍相关API
- `POST /api/self-intro/generate` - 生成自我介绍
- `POST /api/self-intro/get` - 获取自我介绍

#### 3.3.3 智能题库相关API
- `POST /api/question-bank/generate` - 生成题库
- `POST /api/question-bank/get` - 获取题库

#### 3.3.4 模拟面试相关API
- `POST /api/mock-interview/start` - 开始模拟面试
- `POST /api/mock-interview/answer` - 提交面试答案
- `POST /api/mock-interview/end` - 结束模拟面试

#### 3.3.5 面试策略相关API
- `POST /api/strategy/analysis` - 生成画像分析
- `POST /api/strategy/questions` - 生成反问问题
- `GET /api/strategy/analysis/history` - 获取分析历史
- `GET /api/strategy/questions/history` - 获取问题历史
