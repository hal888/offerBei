# Offer贝 (OfferBei)

**AI 驱动的智能面试辅助系统** - 一站式解决简历优化、面试准备、模拟练习等求职难题，助力求职者斩获心仪 Offer！

[![Project Status](https://img.shields.io/badge/status-active-brightgreen.svg)](https://github.com/hal888/offerBei)
[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)

---

## ✨ 核心亮点

- 🤖 **AI 智能分析** - 基于 DeepSeek 大模型，提供专业的简历优化和面试指导
- 🌍 **双语支持** - 完整的中英文界面和内容生成
- 🔐 **用户系统** - JWT 认证，支持注册、登录、密码找回
- 📱 **响应式设计** - 完美适配桌面和移动设备
- 🎯 **个性化服务** - 基于用户简历的定制化面试准备方案

---

## 🚀 核心功能

### 📝 1. 简历解析与优化

**智能分析，精准优化**

- ✅ **多格式支持**：PDF、DOCX、TXT 格式简历上传
- ✅ **深度解析**：AI 自动提取个人信息、教育背景、工作经历、项目经验
- ✅ **智能诊断**：从完整性、匹配度、竞争力三个维度评分
- ✅ **STAR 优化**：基于 STAR 法则（Situation-Task-Action-Result）重写项目经验
- ✅ **关键词注入**：自动识别并建议高频技术关键词
- ✅ **版本管理**：保存原始简历和优化后的简历版本
- ✅ **一键下载**：生成优化后的 PDF 简历

### 🎤 2. 定制化自我介绍

**多版本，多风格，随心选择**

- ✅ **三种时长**：
  - 30 秒电梯演讲版（快速亮点）
  - 1 分钟标准版（平衡覆盖）
  - 3 分钟深度版（详细展开，包含项目细节）
- ✅ **三种风格**：
  - 正式 - 商务专业场合
  - 轻松活泼 - 创业型公司
  - 学术专业 - 技术导向岗位
- ✅ **语音朗读**：TTS 语音合成，支持速度调节
- ✅ **提词器模式**：自动滚动，字体大小/速度可调，深色背景可选
- ✅ **PDF 导出**：保存带格式的自我介绍文档
- ✅ **历史记录**：自动保存，不同版本/风格独立管理

### 📚 3. 智能题库生成

**千人千面的个性化题库**

- ✅ **灵活数量**：10 题（快速练习）/ 30 题（标准练习）/ 50 题（深度准备）
- ✅ **科学分布**：
  - 高频必问题 (30%) - 行业通用问题
  - 简历深挖题 (25%) - 针对个人经历的追问
  - 专业技能题 (25%) - 岗位技术栈相关
  - 行为/情景题 (20%) - BQ 行为面试
- ✅ **话题定制**：支持指定生成话题（如"微服务"、"React 性能优化"）
- ✅ **参考答案**：每题提供高质量参考答案
- ✅ **面试官意图**：解析每个问题背后的考察点
- ✅ **批量导出**：导出完整题库为 PDF
- ✅ **并发生成**：采用批量并发调用，30 题仅需约 30 秒

### 💬 4. 真人模拟面试

**沉浸式面试场景模拟**

- ✅ **面试官风格**：
  - 温和 HR - 友好亲切，适合初次练习
  - 严格技术总监 - 专业严谨，适合技术岗
  - 平衡面试官 - 中性风格，综合练习
- ✅ **时长设置**：15 / 30 / 45 / 60 分钟
- ✅ **实时交互**：
  - 文字模式 - 输入回答，即时反馈
  - 语音模式 - 阿里云 ASR 实时转写（基于 WebSocket）
- ✅ **智能追问**：AI 根据回答内容动态生成后续问题
- ✅ **详细报告**：
  - 多维度评分（专业能力、逻辑表达、自信度、岗位匹配）
  - 逐题分析（问题、回答、反馈、建议）
  - 优化建议汇总

### 📊 5. 面试策略锦囊

**画像分析 + 反问问题，掌握面试主动权**

#### 画像分析
- ✅ **背景信息输入**：职位、经验年数、目标岗位等
- ✅ **优化方向选择**：
  - 空窗期分析与应对
  - 转行背景优势挖掘
  - 经验不足补偿策略
  - 防御性话术准备
- ✅ **结构化分析**：多维度策略建议，每条配有实用 Tips
- ✅ **历史记录**：自动保存，随时查看

#### 反问环节
- ✅ **目标公司/岗位输入**
- ✅ **问题类型筛选**：
  - 公司发展类
  - 团队文化类
  - 岗位职责类
  - 发展机会类
- ✅ **智能生成**：5-8 个高质量反问问题
- ✅ **提问意图解析**：每个问题的背后逻辑和预期效果
- ✅ **PDF 导出**：整合画像分析和反问问题的完整策略文档

### 🌍 6. 国际化支持

**中英双语无缝切换**

- ✅ **界面语言**：中文 / English
- ✅ **智能检测**：
  - 优先级 1：用户手动选择
  - 优先级 2：浏览器语言
  - 优先级 3：默认中文
- ✅ **内容生成**：AI 生成内容自动匹配界面语言
- ✅ **持久化**：语言选择保存到 localStorage，跨页面生效

### 🔐 7. 用户认证系统

**安全可靠的账号体系**

- ✅ **邮箱注册/登录**
- ✅ **JWT Token 认证**
- ✅ **密码强度检测**
- ✅ **忘记密码 / 重置密码**（邮件验证码）
- ✅ **会话管理**：自动登录、记住密码
- ✅ **数据隔离**：不同用户数据完全隔离，支持跨设备同步

### 📖 8. 用户指南

- ✅ **使用手册**：详细的功能说明和使用流程
- ✅ **FAQ 页面**：常见问题解答

## 📦 快速开始

### 环境要求
- Node.js >= 20.19.0
- Python >= 3.8
- MySQL >= 5.7

### 安装依赖

#### 前端依赖
```bash
npm install
```

#### 后端依赖
```bash
cd backend
pip install -r requirements.txt
```

### 配置环境

项目使用 `.env` 文件进行配置管理。

1. **复制配置文件**
   ```bash
   cp .env.example .env
   ```

2. **修改配置信息**
   编辑 `.env` 文件，填入您的实际配置：
   - DeepSeek API Key
   - 数据库连接信息
   - 阿里云 Access Key (用于语音识别)
   - 邮件服务配置

### 初始化数据库
```bash
cd backend
python init_db.py
```

### 启动服务

#### 前端服务
```bash
npm run dev
```

访问地址: `http://localhost:5173`

#### 后端服务
```bash
cd backend
python run.py
```

访问地址: `http://127.0.0.1:5000`

## 📖 使用指南

### 🔐 注册与登录

1. 访问 `http://localhost:5173`
2. 点击右上角「登录/注册」
3. 首次使用需要注册账号：
   - 输入邮箱和密码（密码强度会实时检测）
   - 获取邮箱验证码
   - 完成注册
4. 已有账号直接登录
5. 忘记密码可通过邮箱重置

### 📝 简历优化流程

1. **导航** → 点击「简历优化」菜单
2. **上传** → 选择简历文件（PDF / DOCX / TXT）
3. **分析** → 等待 AI 解析简历（约 10-20 秒）
4. **查看** → 查看智能诊断结果：
   - 简历评分（完整性、匹配度、竞争力）
   - STAR 法则优化建议
   - 关键词优化建议
5. **下载** → 点击「下载优化简历」获取 PDF

### 🎤 自我介绍生成

1. **导航** → 点击「自我介绍」菜单
2. **选择** → 选择版本（30秒/1分钟/3分钟）和风格（正式/轻松/学术）
3. **生成** → 点击「生成自我介绍」（基于已上传的简历）
4. **使用**：
   - 📋 复制文本
   - 🎵 语音朗读（可调速度）
   - 📝 提词器模式（适合练习）
   - 📄 导出 PDF
5**切换** → 可切换不同版本/风格，历史记录会自动保存

### 📚 智能题库使用

1. **导航** → 点击「智能题库」菜单
2. **配置**：
   - 选择题目数量（10 / 30 / 50）
   - （可选）输入自定义话题，如「Spring Boot」
3. **生成** → 点击「生成题库」（30 题约 30 秒）
4. **练习**：
   - 点击「查看答案」查看参考答案
   - 查看「面试官意图」理解考察点
   - 题型按比例分布（高频/深挖/技能/行为）
5. **导出** → 点击「导出题库」生成 PDF

### 💬 模拟面试

1. **导航** → 点击「模拟面试」菜单
2. **配置**：
   - 选择面试官风格（温和 HR / 严格技术总监 / 平衡）
   - 选择面试时长（15 / 30 / 45 / 60 分钟）
3. **开始** → 点击「开始模拟面试」
4. **回答**：
   - 文字模式：直接输入答案
   - 语音模式：点击录音按钮，实时转写
5. **互动** → AI 会根据你的回答智能追问
6. **报告** → 面试结束后查看详细评估报告：
   - 多维度评分雷达图
   - 逐题分析和改进建议

### 📊 面试策略

#### 画像分析
1. **导航** → 点击「面试策略」→「画像分析」
2. **填写** → 输入背景信息（职位、经验、目标岗位）
3. **选择** → 勾选需要的优化方向（空窗期/转行/经验不足/防御话术）
4. **生成** → 点击「生成画像分析」
5. **查看** → 获取多维度分析和应对策略

#### 反问问题
1. **填写** → 输入目标公司和岗位
2. **选择** → 勾选问题类型（公司发展/团队文化/岗位职责/发展机会）
3. **生成** → 点击「生成问题」
4. **使用** → 查看每个问题的提问意图
5. **导出** → 导出完整策略锦囊为 PDF

### 🌍 语言切换

- 点击导航栏右上角的「En / 中」按钮
- 界面和 AI 生成内容会自动切换语言
- 语言选择会保存，刷新页面后保持

## 💡 核心特性说明

### 🔒 数据安全

- **JWT 认证**：基于 Token 的无状态认证
- **密码加密**：使用 Werkzeug 加密存储
- **数据隔离**：不同用户数据完全隔离
- **跨设备同步**：同一账号在不同设备/浏览器数据一致

### ⚡ 性能优化

- **并发生成**：题库批量生成采用并发调用，30 题约 30 秒
- **前端缓存**：localStorage 缓存语言偏好、Token 等
- **懒加载**：路由组件按需加载
- **代码分割**：Vite 自动优化打包

### 🌐 国际化实现

- **前端**：Vue I18n + 智能语言检测
- **后端**：Prompt 模板多语言版本
- **同步机制**：`X-Locale` header 传递语言信息
- **生成内容**：DeepSeek API 根据 locale 生成对应语言
